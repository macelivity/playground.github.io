Text to encrypt







How it works:

The text gets encrypted by taking the ascii value of each of your character from the text and adding the ascii value of your character of the password.

The first character of your text added with the first one of your password, then the next of the text with the next of the password.

When the code reaches the end of the password it uses the first char of the password again and so on until it reaches the end of your text.

Now you only need to encrypt your password into the file to make sure you use the right password when decrypting. Therefore smart people developed

a method called SHA-256 which takes for example a string and converts it into a byte array. Also it works like a one way door:

you can get to the byte array from the string but you can't get the string from the byte array.

That way your password is save and your text also can't be decrypted without the password (and without large computation power).
